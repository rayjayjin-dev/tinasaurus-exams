# 마크다운 기능

Docusaurus는 다양한 마크다운 문법을 지원합니다. 이 문서에서는 자주 사용되는 마크다운 기능들을 시연합니다.

## 제목 레벨

마크다운에서는 `#` 기호의 개수로 제목 레벨을 지정합니다.

### 3단계 제목

#### 4단계 제목

##### 5단계 제목

## 텍스트 서식

다양한 텍스트 서식을 적용할 수 있습니다:

- **굵은 텍스트**: `**굵은 텍스트**`
- *기울임 텍스트*: `*기울임 텍스트*`
- ~~취소선 텍스트~~: `~~취소선 텍스트~~`
- `인라인 코드`: `` `인라인 코드` ``

## 목록

### 순서 없는 목록

- 첫 번째 항목
- 두 번째 항목
  - 중첩된 항목 A
  - 중첩된 항목 B
- 세 번째 항목

### 순서 있는 목록

1. 프로젝트 초기화
2. 의존성 설치
3. 개발 서버 실행
4. 콘텐츠 작성

## 이미지

마크다운에서 이미지를 삽입하는 문법은 다음과 같습니다:

```markdown
![대체 텍스트](이미지_경로 "이미지 제목")
```

## 표 (테이블)

| 기능 | 설명 | 지원 여부 |
|------|------|:---------:|
| 제목 | `#`으로 제목 레벨 지정 | ✅ |
| 목록 | 순서/비순서 목록 | ✅ |
| 코드 블록 | 구문 강조 지원 | ✅ |
| 표 | 테이블 렌더링 | ✅ |
| 수식 | KaTeX 수식 | ✅ |

## 인용문

> Docusaurus는 문서 사이트를 쉽게 유지 관리할 수 있도록 도와줍니다.
> 마크다운 기반으로 콘텐츠를 작성하고, 정적 사이트로 빌드할 수 있습니다.

## 코드 블록 예시

마크다운 문서 내에서 코드 블록을 사용하는 기본 문법입니다:

```javascript
// 간단한 JavaScript 예제
function greet(name) {
  return `안녕하세요, ${name}님!`;
}

console.log(greet('Tinasaurus'));
```
